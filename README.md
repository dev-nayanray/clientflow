# ⚡ ClientFlow AI — Automated Client Acquisition

AI-powered client acquisition: Find leads → Draft emails → Follow up → Send proposals → Book meetings.

## 🚀 Deploy to Vercel (2 minutes)

### Option A: Vercel CLI (fastest)
```bash
npm install -g vercel
cd clientflow
npm install
vercel --prod
```

### Option B: Vercel Dashboard (no terminal needed)
1. Go to https://vercel.com → Sign up free
2. Click "Add New Project"
3. Upload this folder (drag & drop the zip)
4. Click Deploy — done!

### Option C: GitHub + Vercel (auto-deploy)
1. Push this folder to a GitHub repo
2. Go to vercel.com → Import Git Repository
3. Select your repo → Deploy

## 🔑 API Key Setup
After deploying, enter your Anthropic API key in the app's banner.
Get your key at: https://console.anthropic.com

## 📦 Local Development
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Features
- ⚙️ Setup tab: configure niche, country, service, pricing, Calendly link
- 🚀 Workflow tab: one-click runs all 5 stages with live progress
- 👥 Leads tab: view leads, generate personalized emails/proposals/meetings per lead
